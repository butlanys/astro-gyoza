* 自带一键部署脚本
* 自带签名服务器(qsign-1.1.7)
  ============使用教程=================
  下载云盘内的ZeroTermux和恢复包
* 先安装ZeroTermux
* 把恢复包直接解压在内部存储（防止你们不会放）

打开ZeroTermux，在左侧边栏中找到备份/恢复，然后选择恢复数据包
![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/0.jpg)

* 选择以后会弹出恢复的系统名字，填一个你喜欢的就行。
* 恢复完成后在左侧边栏找到容器切换，切换到你刚才恢复的容器
  ============bot使用教程==============
  使用方法：
* 终端运行

```
bash 云崽bot容器启动脚本.sh
```

![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/1.jpg)

* 选择 debian-bullseye_arm64 _1--proot
  ![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/2.jpg)
* 选择 启动proot debian-bullseye_arm64
  ![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/3.jpg)
* 终端运行

```
./start.sh
```

启动签名服务、redis和查看使用说明
![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/4.jpg)
最后再输入

```
cd Miao-Yunzai
```

切换到云崽目录
然后运行pnpm app登陆Bot
![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/5.jpg)

运行后你将会看到跟上面一样的内容

====================================
第一次运行可能会出现连接不上签名服务器，请用Ctrl+c结束掉bot进程，然后用ps -le查看正在运行的服务，如果正常，你将看到以下内容
![](https://alist.hqycloud.top/d/onedrive-oss/img/yunzaibot-0/6.jpg)

* 里面包含一个java程序和redis数据库服务
  ===========pnpm常用命令=============

```
pnpm app #带log启动
```

```
pnpm run log #查看启动日志
```

```
pnpm start #不带log启动
```

```
pnpm stop #停止bot
```

以上命令请在云崽bot主目录下运行
