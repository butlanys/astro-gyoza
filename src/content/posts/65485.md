前阵子因为某些原因需要新建个用户，今天连接ssh提示：



```CMD
C:\Users\butla>ssh alice
Bad permissions. Try removing permissions for user: redmiG\\butlanys1 (S-1-5-21-) on file C:/Users/butla/.ssh/config.
Bad owner or permissions on C:\\Users\\butla/.ssh/config
```

解决方法：

打开.ssh目录我这里是C:/Users/butla/

右击.ssh文件夹——属性——安全——下方的高级——点击下方取消继承——选择第一项

取消掉所有用户权限后，为当前用户单独赋权

还是右击.ssh文件夹——属性——安全——下方的高级——添加——选择主体——高级——立刻查找——找到当前用户——添加——一路确定

然后在终端重新输入ssh连接命令

```CMD
C:\Users\butla>ssh alice
Welcome to Ubuntu 22.04.5 LTS (GNU/Linux 6.5.0-41-generic x86_64)
```

完！
