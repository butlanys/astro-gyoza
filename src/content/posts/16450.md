# 一、前言

此教程适用自行开发网页、博客，以及有自定义aplayer需求的群体

比如说[aplayer播放器对接alist | butlanys的博客](https://blog.hqycloud.top/posts/51355/)播放等 ~~（此文章正在写，稍安毋操）~~

# 二、准备

下载css、js到自己的网站目录下

地址：[APlayer/dist at master · DIYgod/APlayer](https://github.com/DIYgod/APlayer/tree/master/dist)

致使用hexo butterfly主题的站长们，如果已在博客中使用aplayer需要在配置文件中把aplayerInject里enable值改为false

# 引入

这里以hexo butterfly为例

在_config.butterfly.yml文件的inject—head中加入

```css
<link rel="stylesheet" href="/pluginsSrc/aplayer/dist/APlayer.min.css">
```

请将路径改为你自己的路径

在bottom中加入

```javascript
<script src="/pluginsSrc/aplayer/dist/APlayer.min.js"></script>
```

# 调用

新建一个js文件

并参照官方文档进行修改：[首页 - APlayer](https://aplayer.js.org/#/zh-Hans/?id=%E5%85%A5%E9%97%A8)

# 引入自定义的配置文件

回到_config.butterfly.yml中，在bottom字段添加

```javascript
<div id="aplayer"></div>
<script src="自定义的js文件名.js"></script>
```

# 完成

接下来就可以刷新你的网页，查看是否出现aplayer播放器
